EL RETO

Los sistemas de iluminación LED para acuarios disponibles en el mercado son caros, propietarios y no permiten personalización. El objetivo era construir una solución desde cero que permitiera control total del espectro lumínico adaptado a las necesidades de cada especie acuática.

STACK TÉCNICO

  • HardwareESP32 + Módulos LED PWM de alta potencia
  • FirmwareArduino IDE (C++)
  • InterfazApp móvil React Native
  • ProtocoloWebSocket en tiempo real
  • Diseño 3DFusion 360 + Impresión FDM

SOLUCIÓN

Diseño y fabricación de una lámpara LED de espectro completo controlada por ESP32 con comunicación WiFi. La interfaz web permite configurar ciclos de amanecer/atardecer, intensidad por canal RGB+W y perfiles personalizados guardados en memoria.

APRENDIZAJE

  • 01Control PWM multi-canal con ESP32
  • 02Diseño e impresión 3D de disipador de calor
  • 03Comunicación WebSocket hardware-software
  • 04Gestión térmica en electrónica de potencia

CONCLUSIÓN

Este proyecto demostró que es posible construir hardware profesional de alta precisión con componentes de bajo coste y código abierto. La combinación de ESP32 con PWM multi-canal y una interfaz web reactiva ofrece una experiencia de usuario comparable a productos comerciales de gama alta, a una fracción del precio.
